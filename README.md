# Supply Chain Management System (Database Project)

This is a complete database project for a **Supply Chain Management System** developed for the Database Management System course at Riphah International University.

## 📁 Files in this repository

| File | Description |
|------|-------------|
| `schema.sql` | Contains all `CREATE TABLE` statements to build the database schema. |
| `sample_data.sql` | Contains `INSERT` statements to populate tables with realistic sample data. |
| `views.sql` | Defines three useful views: current stock, low stock alert, and supplier performance. |
| `queries.sql` | Shows all important `SELECT` queries: joins, subqueries, aggregations, and reports. |

## 🛠️ How to run

1. Use Oracle SQL (Express Edition, SQL Developer, or Oracle Live SQL).
2. Run `schema.sql` first to create tables.
3. Run `sample_data.sql` to insert data.
4. Run `views.sql` to create views.
5. Run `queries.sql` to test the queries.

## 📊 Features

- Normalized database up to 3NF
- One-to-many and many-to-many relationships
- Automatic low‑stock alert
- Business reports: best‑selling products, supplier ranking, warehouse utilization
